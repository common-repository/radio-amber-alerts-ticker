This widget will place a Radio Amber Alert Ticker in your Wordpress sidebar.

== Description ==

This widget adds a Radio Amber Alert Ticker with a photo of the abducted child in your wordpress sidebar. The photo links to the actual Radio Amber Alert page on RadioAmber.net with all the abduction information + photos + recorded mp3 alert. It additionally links to the widget/ticker download page for others to get any of the available tickers. Pages open in a new window, user remains on your site.



    
== Installation ==



1. Extract all files and upload the AmberAlerts folder` to the /wp-content/plugins/ directory. 
2. Activate the plugin through the Plugins menu in WordPress.
3. Go to your Admin panel and choose Appearance, then Widgets and drag the Amber Alerts Ticker to your sidebar.
